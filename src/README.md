# Как работает МЕНЕДЖЕР ПРОДАЖ

## Класс Main
В классе Main ~~творится~~ происходит следующее:
1. Создает массив *sales*
2. Создает новый объект класса SalesManager
3. Выводит наибольшее число из массива

## Класс SalesManager
В классе SalesManager происходит следующее:
1. Конструктор принимает массив чисел и записывает его в поле
2. Метод *max* определяет наибольшее число из массива и возвращает его.
